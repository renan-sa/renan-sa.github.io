Semana OmniStack 11 - Rocketseat

Workshop totalmente gratuito realizado pela Rocketseat para apresentar a stack NodeJs, React, React Native e Expo em um projeto para desenvolver uma aplicação web e mobile. A aplicação se chama Be the Hero e é um portal para ONGs se cadastrarem e lançarem suas ações e as despesas, e os cidadãos via aplicação mobile escolhem uma das despesas e entram em contato para se colocar a disposição ou auxiliar financeiramente.
